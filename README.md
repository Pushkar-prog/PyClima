# 🌍 PyClimaExplorer

**A Streamlit-based interactive climate intelligence dashboard** built with synthetic CESM1-style data (1920–2018). Features 8 interactive tabs: heatmaps, time series, trend maps, El Niño detection, year comparison, time-lapse animation, story mode, and a hackathon pitch deck.

---

## 🖥️ Screenshots

The dashboard uses a **white Canva KPI style** with Space Grotesk headings, colored KPI cards, and clean Plotly charts on a white background.

---

## 📦 Requirements

- Python **3.10+**
- pip (comes with Python)

---

## 🚀 How to Run

### Step 1 — Clone / open the project folder

```bash
cd c:\pyclima
```

### Step 2 — Create a virtual environment (recommended)

```bash
python -m venv .venv
```

Activate it:

```bash
# Windows (PowerShell)
.venv\Scripts\Activate.ps1

# Windows (CMD)
.venv\Scripts\activate.bat
```

### Step 3 — Install dependencies

```bash
pip install -r requirements.txt
```

### Step 4 — Generate the data and all app components

Run the four phase scripts **in order** (only needed once):

```bash
python phase1_bootstrap.py
python phase2_features.py
python phase3_polish.py
python phase4_final.py
```

> **What these do:**
> - `phase1_bootstrap.py` — generates `data/cesm.zarr` (synthetic CESM1-style climate data)
> - `phase2_features.py` — adds core dashboard features
> - `phase3_polish.py` — generates `static/story_mode.html`
> - `phase4_final.py` — final audit and polish

### Step 5 — Run the app

```bash
streamlit run app.py
```

The app will open automatically at **http://localhost:8501** in your browser.

## 🗂️ App Tabs

| Tab | Description |
|-----|-------------|
| 🗺️ **Heatmap** | Interactive global climate heatmap (5 variables, season + year filter) |
| 📈 **Time Series** | Annual trends + 5yr/10yr rolling means at any grid point |
| 🔥 **Trend Map** | Linear warming/cooling trend per decade across 98 years |
| 🌊 **El Niño Detector** | Auto-detects ENSO events using Niño 3.4 index |
| 🔄 **Comparison** | Side-by-side diff map between any two years |
| ▶️ **Time-lapse** | 98-frame animated playback across 1920–2018 |
| 📖 **Story Mode** | Cinematic 4-chapter guided tour |
| 🎤 **Pitch Mode** | Timed hackathon demo script + Judge Q&A |

## 📝 Notes

- The dataset is **synthetic** but statistically parameterised from CESM1 Large Ensemble properties: +1.1 °C total trend over 98 years, ~30 ENSO cycles, realistic spatial structure.
- To swap in **real data**: replace `data/cesm.zarr` with a real NetCDF/Zarr file in the same schema — no other code changes needed.
- The `data/cesm.zarr` directory is auto-regenerated if missing, so it is safe to delete and re-run.
